# Snake Game

A classic **Snake** game written in Python using the `pygame` library. Control the snake, collect apples to score points and grow longer, but be careful not to hit the walls or your own body!

## Features

* **Classic Gameplay:** Collect apples to grow your snake and increase your score.
* **Custom Graphics:** The game features custom textures for backgrounds, banners, and items. The snake has animated eyes that change position depending on the direction of movement.
* **Sound Effects & Music:**
  * Background music plays during gameplay.
  * Custom sound effects for eating apples, hitting obstacles, and a special "Game Over" tune.
* **Interactive Pause Menu:**
  * Triggered by pressing the `ESC` key or when the game is over.
  * Options to start a `New game` or `Quit` the application.
  * A clickable MUTE button to toggle the background music on and off.

## Controls

* **W** - Move Up
* **S** - Move Down
* **A** - Move Left
* **D** - Move Right
* **ESC** - Pause Game / Open Menu
* **Left Mouse Button** - Interact with the pause menu (buttons and mute toggle)
